Package com.github.amkay.gradle.gitflow.version

Contains the object that is set as version property on the project to which this plugin is applied. The most important aspects of this version object are:
  • The version is lazily inferred. That means the plugin only interacts with Your Git repository if a method on this version object is called, e.g. toString() . This functionality is realized by DelayedVersionWithType, which is the actual type of the set version object.
  • This DelayedVersionWithType delegates to VersionWithType, which provides the real functionality of the version object.
  • The class VersionWithType actually delegates to Version and provides additional functionality to determine the type of the inferred version.
